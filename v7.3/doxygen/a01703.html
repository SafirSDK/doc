static std::wstring Safir::Control::Parameters::RebootCommand ( )

Command to be executed for a computer reboot initiated by Safir.

The command must be given as an absolute path. Most commonly this is a operating system specific command, for example '/usr/bin/shutdown –reboot now' for Linux or 'C:\WINDOWS\System32\shutdown /r /t 0' for Windows. Note that it might be necessary to make additional configurations outside Safir to be able to run the command.
